서비스 소개

IT인재 검증 서비스 PBT에 대해 소개합니다.

축척된 기업 채용 모델 데이터를 이용해
취업매칭 만족도를 높입니다.

PBT는 역량측정과 실무 프로젝트 미션을 통해
검증된 IT인재를 발굴하는 서비스입니다.

PBL vs PBT

PBL PBT
학생 구직자
Project Based Learning 프로젝트를 진행하며 관련 지식을 학습하는 방식입니다. Project Based Test 기업맞춤형 실무 프로젝트로 구성된 미션들을 해결하며 구직자의 실력을 검증합니다.

구인사이트 vs PBT

구인사이트 PBT
구직자가 올린 이력서를 채용담당자가 확인합니다. 검증에 많은 시간과 과정이 필요합니다. 구직자가 제출한 미션 결과물을 전문가와 AI엔진이 분석합니다. 검증된 리포트로 채용 시간과 비용을 줄일 수 있습니다

매직에꼴 PBT는 3단계를 거쳐
지원자를 검증합니다.

Step 01. 역량진단

IT분야에 대한 역량을 측정하고
개개인의 성향과 잠재역량을 파악합니다.

# 실무역량 # HR역량

Step 02. 프로젝트 미션 수행

미션 결과를 기반으로 실무 수행 능력을 평가하고 역량 및 성향 데이터를 추출합니다.

# 실무프로젝트 # 역량데이터화

Step 03. 분석 및 매칭

데이터와 실무자가 분석한 내용을 바탕으로 회사에 적합한 인재를 추천합니다.

# 인재분석 # 인재추천
